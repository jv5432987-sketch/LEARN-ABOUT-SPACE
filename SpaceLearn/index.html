About SpaceLearn

SpaceLearn is a lightweight Progressive Web App (PWA) made to teach space science in a clean, fun way. After opening once, it works offline.

  • Install with the Install button in the header.
  • Switch tabs to explore content.
  • Everything is stored locally on your device.

Built with vanilla HTML/CSS/JS for maximum compatibility.
